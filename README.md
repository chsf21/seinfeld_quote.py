# seinfeld_quote.py
A simple Python script using BeautifulSoup that grabs a random Seinfeld quote from seinfeldism.com and displays it in the terminal.  
Additional features:  
* Optionally display link to quote on seinfeldism.com (useful for sharing with friends).
* Quotes are not repeated during one session until the user has read through the entire collection. This provides for a more enjoyable experience. 

Requires python3 and BeautifulSoup version 4.  
Download BeautifulSoup version 4: https://www.crummy.com/software/BeautifulSoup/#Download
